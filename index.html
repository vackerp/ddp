The app

This very basic shiny app is intended for the Coursera Course 'Developing Data Products'. It's a very basis BMI-calculator. The app is hosted on shinyapps.io.

It need 2 parameters: The height (in meters en centimeters (example: 1.78) and the weight (in kg (example: 75))

Output: the MBI-index. Calculation:

height = 1.78
weight = 75
weight / height^2
